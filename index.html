C++ Inventory & Profit Projection Software

VIEW CODE AT GITHUB

This is a program that allows the user to: create and save an inventory, add items, search for items, and check the projected profit from selling however many units of each item. The user can also create a transaction file which modifies the values of each selected item. The transaction file is then read back in to modify the inventory. There is no GUI for this program, as it was one of the earlier programs I created.
