Java FX GUI Card Game With AI

This is a fully functioning card game I developed as a CS project, that uses some java FX animations. The opponent is a fully functioning AI script that makes decisions based on various circumstances.

Java FX GUI Inventory Program

This is an inventory program that allows the user to add/remove items and sort the inventory by title or date loaned. It saves the inventory to a binary file called inventory.bin. The user can loan items and mark them as returned after the item is returned to the user's inventory.
